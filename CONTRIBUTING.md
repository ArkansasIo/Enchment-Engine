# Contributing to Enchentment Engine

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository and create your branch from `main`.
2. Make your changes, following Rust best practices and existing code style.
3. Add tests and documentation as needed.
4. Run `cargo build` and `cargo test` to ensure everything works.
5. Submit a pull request with a clear description of your changes.

## Code Style
- Use idiomatic Rust and keep modules organized.
- Document public APIs and major systems.
- Prefer modular, extensible code.

## Reporting Issues
- Use GitHub Issues for bugs, feature requests, and questions.

## Getting Help
- See the README and code comments for guidance.
- For major changes, open an issue to discuss your proposal first.

Happy coding!
