# Inventory & Item Systems Documentation

This section covers the tools and systems for managing items, equipment, and inventory in an RPG engine.

## Item Tools
- **Item Database Editor**: Centralized tool for creating, editing, and organizing all in-game items. Supports categories, rarity, stats, and descriptions.
- **Equipment Slot Manager**: Defines valid equipment slots (main hand, off hand, head, chest, etc.) and enforces slot rules for each item type.
- **Crafting & Recipe Editor**: Allows designers to create crafting recipes, define required ingredients, and set crafting outcomes.
- **Loot Table Generator**: Configures drop rates, loot pools, and rarity for enemies, chests, and events.
- **Currency & Economy System**: Manages in-game currencies, shops, trading, and economic balance.

## Item Types
- Weapons, Armor, Shields
- Consumables (potions, food, scrolls)
- Quest Items
- Materials & Crafting Components
- Relics, Artifacts, and Unique Items

## Best Practices
- Use data-driven item definitions for easy balancing and modding.
- Support stackable and unique items.
- Provide search, filter, and sort features in the inventory UI.
