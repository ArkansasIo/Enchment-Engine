# Usage Guide: Enchentment Engine

## Quick Start
- Build: `cargo build --release`
- Run demo: `cargo run --example demo_game`
- Add your own game in `examples/`

## Engine Features
- Modular plugin system
- Keybinds and macros
- Asset management
- Map, AI, and core systems

## Example Code
```rust
use engine::keybinds::*;
use engine::macros::*;
// ...
```
