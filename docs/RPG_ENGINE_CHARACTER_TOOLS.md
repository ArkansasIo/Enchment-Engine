# Character & Creature Tools Documentation

This section details the tools and systems for creating, editing, and managing characters, creatures, and NPCs in an RPG engine.

## Character Systems
- **Character Creator**: Allows players or designers to customize appearance, gender, race, class, and background.
- **Race & Class Templates**: Define base stats, abilities, and traits for each race and class. Supports hybrid and custom classes.
- **Equipment Preview System**: Visualizes equipped items, armor, and weapons on the character model.
- **Stat & Attribute Editor**: Edits core stats (strength, agility, intelligence, etc.), derived stats, and growth curves.

## Animation & AI
- **Animation Retargeting**: Adapts animations to different character rigs and body types.
- **Behavior Tree Editor**: Visual tool for designing complex AI behaviors for NPCs and creatures.
- **Dialogue & Interaction Rules**: Sets up conversation trees, branching responses, and interaction triggers.
- **Schedule Routines**: Assigns daily routines, sleep cycles, and work schedules to NPCs.
- **Faction & Reputation Logic**: Manages relationships, alliances, and enmity between characters, factions, and the player.

## Best Practices
- Modularize character data for easy extension (e.g., new races, classes, or skills).
- Use visual editors for AI and dialogue to speed up iteration.
- Support both player and non-player character creation.
