# Enchentment Engine Modules

This document describes each major module/crate in the engine.

## TheFramework
Handles window creation, user events, and UI system.

## Rusterix
Core game engine, software rasterizer, game logic.

## SceneVM
Layer-based renderer, GPU rendering, main graphics backbone.

## Creator
Editor application for game development.

## Client
Game runtime for players.

For API and implementation details, see the respective crate folders.
