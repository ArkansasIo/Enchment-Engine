# Enchentment Engine Overview

Enchentment Engine is a next-generation classical RPG creation suite. It enables developers to build 2D, isometric, and first-person RPGs with modern features.

## Features
- 2D, isometric, and first-person RPG creation
- Modern multiplayer support
- Procedural content generation
- Modular, extensible engine
- Flexible rendering (software & GPU)
- Cross-platform
- Custom UI and event system
- Open-source (MIT License)
