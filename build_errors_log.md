# Enchentment Engine Error & Issue Log

This log tracks errors, warnings, and other issues encountered during development, build, or runtime. Use this file to document problems, their status, and resolutions for future reference and debugging.

---

## How to Use
- Add a new entry for each error, warning, or issue.
- Include date, file/module, error/warning message, and status (open/resolved).
- Optionally add steps to reproduce, screenshots, or links to related issues/commits.

---

**Date:** 2026-02-16  
**Type:** Error  
**File/Module:** creator/src/editor.rs  
**Message:** "Failed to load project config: file not found"  
**Status:** Open  
**Resolution:** (pending)
