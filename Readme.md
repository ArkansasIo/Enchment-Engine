# Enchentment Engine

Enchentment Engine is a modular RPG/MMORPG game engine and editor in Rust with:
- 2D/3D editing workflows
- Unreal-style IDE layout (top/left/right toolbars, menus, windows)
- Town generation inspired by TownGeneratorOS
- Fantasy world generation (continents, countries, capitals)
- RPG/MMORPG data generation and simulation

## Quick Start

```bash
cargo build -p enchantmen-creator
cargo run -p enchantmen-creator
```
